# borderline 图片样本

存放难以直接归类、存在明显权衡或会因场景变化而改变判断的边界图片。

## 筛选标准

- 同时存在清晰的优点和缺点。
- 判断依赖受众、渠道、目标或上下文。
- 多次复核可能产生不同结论，适合用于校准规则。

## 命名与维护

- 命名格式：`borderline_日期_主题_序号.扩展名`，例如 `borderline_20260610_商品主图_001.png`。
- 在相关记录中补充来源、授权情况、争议点、适用条件、标签和复核日期。
- 定期复核；判断稳定后移入 `good/` 或 `bad/`。

后续检索或 Agent 应优先使用本目录理解判断边界与不确定性。
