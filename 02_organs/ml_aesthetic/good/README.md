# good 图片样本

存放明确符合个人审美与任务目标的正例图片。

## 筛选标准

- 多次复核后仍能明确说明“好在哪里”。
- 与目标场景匹配，不仅是单纯画面漂亮。
- 能对应至少一条 `visual_rules.md` 中的正向规则。

## 命名与维护

- 命名格式：`good_日期_主题_序号.扩展名`，例如 `good_20260610_商品主图_001.png`。
- 在相关记录中补充来源、授权情况、判断理由、标签和复核日期。
- 重复或过时样本应删除或移至更合适分类。

后续检索或 Agent 可将本目录作为正例参考，但必须结合规则和任务上下文。
