## 核心内容

- Progressive Disclosure 是标准 UX 术语，指先展示当前任务必需信息，把高级选项和复杂度延后暴露。
- Agent Workspace 指 Agent 不只是聊天框，而是带文件、结果、工具、历史和上下文的工作区。
- Human-in-the-loop 指关键节点保留人工确认、审核、编辑或决策，而不是完全自治。
- Multimodal Canvas 指把文本、图片、视频、文件与 Agent 结果放在统一空间中组织和编排。
- Tool Orchestration 或 Agentic Workflow 指多工具、多步骤、多 Agent 的调度、执行和结果衔接。

## 边界与例外

- 这些术语适合用于产品方案和团队沟通，但不自动决定最终信息架构和实现方案。
- Copilot、Agent、Workspace、Canvas 在不同团队语境下边界会漂移，落到产品前仍需重新定义。
- 如果最终产品强调确定性流程而不是开放探索，workflow 和 orchestration 的比重会高于 workspace 叙事。

## 失效风险

中。术语本身常见，但 AI 产品行业的偏好和热词变化较快。
