## 输入要求

- 风格关键词

## 提示词正文

```text
你是一个高级前端开发工程师。任务：基于以下参考和数据，设计一个个人作品集网站。

## 必须提供
1. **参考网站**：2-3个截图路径或URL，说明你喜欢它们的哪些具体设计元素
2. **内容数据**：名字、定位、项目列表、技能列表
3. **风格关键词**：3-5个具体的风格描述词（不要只说'Linear风格'，要说'微妙渐变光晕、超大标题排版、克制的hover动画'）

## 设计要求
- 从用户身份出发设计概念，不要套模板
- 每个设计决策都要有理由
- 必须先读取参考截图分析设计模式
- 完成后必须 `npm run build` 验证
```

## 输出格式

完整的Next.js项目，可直接npm run dev预览

## 成功标准

- 用户认可视觉效果
- 有独特设计概念（不是通用模板）
- 构建成功
